# 🧠 MathPRO AI Assistant
### Personalized for Mahamudul Hasan (ADUST)

An ultra-fast, modern Mathematics Assistant powered by **Google Gemini 1.5 Flash**. Built with **Next.js 14**, **Tailwind CSS**, and **KaTeX** for high-performance math rendering.

## 🚀 Features
- **Step-by-Step Solutions:** Solves complex calculus, algebra, and discrete math.
- **Ultra-Fast:** Optimized with Gemini Flash for instant responses.
- **Premium UI:** Dark-themed, glassmorphic design tailored for ADUST students.
- **Mobile Responsive:** Study on the go with a fully responsive interface.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **AI Model:** Google Gemini 1.5 Flash
- **Styling:** Tailwind CSS + Lucide Icons
- **Math Rendering:** KaTeX + React Markdown
